# ddns-ukraine.com.ua-js

JS script to update ip-address at ukraine.com.ua hosting.

# Getting started

## Requirements:

    node v6.9.1
    npm v3.10.8

## Install:

    npm install

## Configure

Create `config.json` and fill it with correct data

    cp config.json.template config.json

## Run

    npm start
